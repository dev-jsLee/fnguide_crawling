# FnGuide 크롤링 프로젝트 리팩토링 계획

## 1. 프로젝트 구조 개선
- [ ] 프로젝트 구조 재정립
    ```
    fnguide_crawling/
    ├── src/
    │   ├── core/                 # 핵심 비즈니스 로직
    │   │   ├── crawler/         # 크롤링 관련 모듈
    │   │   ├── parser/          # 데이터 파싱 모듈
    │   │   └── validator/       # 데이터 검증 모듈
    │   ├── gui/                 # GUI 관련 모듈
    │   │   ├── main_window.py   # 메인 윈도우
    │   │   ├── dialogs/         # 다이얼로그
    │   │   └── widgets/         # 커스텀 위젯
    │   ├── utils/               # 유틸리티 함수
    │   └── config/              # 설정 관리
    ├── tests/                   # 테스트 코드
    ├── docs/                    # 문서화
    └── scripts/                 # 실행 스크립트
    ```
